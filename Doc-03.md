# マンガアクセシビリティ・トピックノート

## この文書について

この文書は、現時点では十分に整理されていないものの、今後のラウンドテーブルで重点的に議論したいテーマを記録するためのノートです。

ロードマップとの違いは、ここで扱う内容はまだ結論や整理方法が定まっていないことです。

そのため、この文書には未整理の論点や仮説も含まれます。

将来的には、それぞれの項目がロードマップへ統合されたり、独立した研究テーマへ発展したりすることを想定しています。

# 現時点で優先して検討したいテーマ

## 1. マンガ表現と構造化データ

マンガの表現を、どこまでデータとして記録できるのか。

また、記録すべきなのか。

例えば、

- コマの読み順
- 視線誘導
- コマ内の時間表現
- 見開き全体の演出
- ページをまたぐ構成

などは、アクセシビリティだけでなく、マンガ表現そのものにも関わるテーマです。

海外ではEPUBやReadiumなどで関連する提案が行われていますが、日本のマンガ表現との適合性については、今後あらためて検討する必要があります。

## 2. AIとマンガ読解

AIは、マンガをどこまで理解できるのでしょうか。

また、人間が持つマンガ読解の知識を、どのようにAIへ反映できるのでしょうか。

例えば、

- マンガの文法
- 作家ごとの特徴
- ジャンルごとの表現
- 文脈を踏まえた解説
- 人間とAIの役割分担

などは、今後の重要な研究テーマになると考えています。

## 3. ラウンドテーブルの運営

ラウンドテーブルそのものについても、今後検討すべき課題があります。

例えば、

- 参加者の構成
- 情報共有の方法
- 研究成果の公開
- 出版社との連携
- 学会との関係
- 継続的な運営体制

などです。

これらについても、今後少しずつ整理していく予定です。



# 今後検討したいテーマ

ここで整理するテーマは、現時点では十分な結論を持っていません。

しかし、マンガアクセシビリティを考える上で避けて通れない論点であり、今後のラウンドテーブルで継続的に議論したい項目です。

## 4. 標準化と日本のマンガ表現

海外では、電子出版やアクセシビリティの分野で、作品構造や読み順を記録するための標準化が進められています。

一方、日本のマンガは、

- 見開きを一つの演出単位として用いること
- コマ内に複数の時間を表現すること
- 作家ごとに大きく異なる視線誘導
- 読み順が一意に定まらない演出

など、多様な表現を持っています。

今後は、

- 海外仕様を日本のマンガへ適用できるのか
- 日本独自に考慮すべき表現は何か
- 日本から国際的な標準化へ提案できることは何か

を整理する必要があります。

## 5. 出版社における実装方法

アクセシビリティを実現するためには、

「誰が、いつ、どの工程で情報を作るのか」

という実務上の課題があります。

例えば、

- 編集工程で付与するのか
- 出版後に付与するのか
- AIが自動生成するのか
- 人間が確認するのか

など、制作フローそのものについても検討が必要になります。

## 6. 国際的な動向

電子出版、アクセシビリティ、AIなどの分野では、海外でも議論が進んでいます。

今後は、

- 国際標準
- 海外の研究事例
- 海外出版社の取り組み
- 各国のアクセシビリティ政策

なども継続的に調査し、日本との違いや共通点を整理していく必要があります。

## 7. 長期的な研究体制

マンガアクセシビリティは、短期間で結論を出せるテーマではありません。

そのため、

- 共同研究
- 実証実験
- 学会活動
- 出版業界との連携
- 当事者参加
- 若手研究者・技術者の育成

なども視野に入れた継続的な研究体制について検討していきます。
