# マンガアクセシビリティ・参考資料集

## この文書について

この文書は、ラウンドテーブルやロードマップを検討する際の参考資料を整理するための文書です。

ここでは、国内外の規格、研究、実証実験、業界動向などを継続的に整理していきます。

現時点では、網羅的な文献一覧ではありません。

まずは、今後の議論で重要になると思われる分野を整理し、必要に応じて情報を追加していきます。

# 今後重点的に調査したいテーマ

この文書は、単なる文献一覧ではありません。

「今後、どの分野を重点的に調査するか」を整理するためのメモでもあります。

現時点では、次のテーマを優先的に調査することを想定しています。

## 1. 国際標準化の動向

電子出版やアクセシビリティに関する国際的な標準化の動向を継続的に調査します。

主な調査対象

- W3C
- DAISY Consortium
- EPUB Working Group
- Readium Foundation
- EDRLab
- その他関連団体

調査の目的

- マンガに関する議論がどこまで行われているか
- 日本のマンガ文化との共通点と相違点
- 日本から提案できるテーマの整理

## 2. 国内の出版業界

出版社や電子コミック配信事業者の取り組みを整理します。

主な調査対象

- 出版社
- 電子書籍制作会社
- 電子コミック配信事業者
- 業界団体
- 関連研究会

調査の目的

- 現在の制作工程
- アクセシビリティ対応の現状
- 技術的・制度的課題

## 3. マンガ研究

マンガ表現に関する研究を継続的に調査します。

主な調査対象

- コマ割り
- 視線誘導
- 時間表現
- 見開き演出
- ジャンル研究
- 作家研究

調査の目的

- 音訳・解説との関係
- AIへの知識利用
- 構造化との接点

## 4. AI・情報技術

AI技術の進展を継続的に確認します。

主な調査対象

- マルチモーダルAI
- コンピュータビジョン
- OCR
- 音声合成
- 自然言語生成
- エージェント技術

調査の目的

- 解説生成への応用
- 人間との役割分担

## 5. アクセシビリティ

読書バリアフリーや情報アクセシビリティの動向を調査します。

主な調査対象

- 読書バリアフリー
- DAISY
- 音訳
- 音声解説
- 利用者調査
- 支援技術

調査の目的

- マンガとの共通課題
- 利用者要求の整理
- 他分野から応用できる知見

# 文献整理の基本方針

参考資料は、単に収集することを目的としません。

それぞれの資料について、

- 何が提案されているか
- マンガアクセシビリティとの関係
- 今後の課題
- ラウンドテーブルで議論すべき点

を整理しながら蓄積していきます。

今後は、資料ごとに要約やコメントを付与し、この文書自体も継続的に更新していく予定です。
