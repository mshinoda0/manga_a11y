# マンガアクセシビリティ・ラウンドテーブル構想

## はじめに

この文書は、マンガアクセシビリティについて議論するための「たたき台」です。

完成した提案でも、特定の結論を示すものでもありません。

目的は、マンガアクセシビリティというテーマを、特定の専門分野だけで考えるのではなく、多様な立場の人々が共通の地図を見ながら議論できる状態をつくることです。

現在、この分野では、出版社、研究者、視覚障害当事者、エンジニア、電子出版関係者など、それぞれが個別に課題へ取り組んでいます。

しかし、マンガというメディアは、

- 表現
- 出版
- 電子書籍
- アクセシビリティ
- AI
- 法律
- 標準化

といった複数の分野が重なり合うため、一つの専門分野だけでは整理しきれません。

本構想では、この状況を「それぞれの専門家が、自分のレンズから同じ作品を見る場」と考えます。

重要なのは、誰か一人が正しい答えを持つことではなく、それぞれが持っている知識や経験を持ち寄り、相互に補完しながら議論を進めることです。

## この構想が目指すもの

本構想が目指すのは、新しい規格やルールを直ちに作ることではありません。

まず必要なのは、

「今、どのような課題が存在し、誰の知見が必要なのか」

を共有することです。

その上で、

- すでに取り組まれていること
- まだ誰も取り組んでいないこと
- 海外で議論が始まっていること
- 日本独自に検討すべきこと

を整理し、中長期的なロードマップを描いていくことを目的とします。

この文書は、その出発点となる作業文書です。

今後の議論を通して内容を更新し、必要に応じて修正・追加を行うことを前提としています。




# 参加を想定する人たち

マンガアクセシビリティは、一つの専門分野だけで解決できる課題ではありません。

むしろ、それぞれ異なる立場の人たちが、互いの専門性を理解しながら議論することに価値があります。

現時点では、次のような分野からの参加を想定しています。

- マンガ研究者
- 出版社（編集・制作・電子出版・法務）
- 電子コミック配信事業者
- 標準化・電子出版技術の専門家
- AI・画像認識・情報工学の研究者、エンジニア
- 視覚障害当事者
- 音訳・アクセシビリティ実務者
- 図書館・教育・読書バリアフリー関係者
- 必要に応じて行政・業界団体・関連学会

ここで重要なのは、誰か一人が全体を理解していることではありません。

例えば、マンガ研究者は作品表現について深い知見を持っています。一方、出版社は制作工程や流通を理解しています。エンジニアは実装方法を考え、視覚障害当事者は実際の利用体験を語ることができます。

それぞれが異なる「レンズ」を持ち寄ることで、初めて全体像が見えてきます。

# 当面の進め方

本構想では、最初から大きな組織を作ることは考えていません。

まずは、小規模な情報共有と意見交換を重ねながら、論点を整理することを優先します。

当面は、次のような進め方を想定しています。

1. 現状の課題を整理し、共通のロードマップを作成する。

2. 必要に応じて専門家へ個別に意見を求める。

3. 小規模なラウンドテーブルを開催し、異なる立場から自由に議論する。

4. その成果を整理し、次の課題や実証実験につなげる。

現時点では、結論を急ぐことよりも、「議論できる状態」を作ることを重視します。



# 本構想の基本姿勢

この構想には、あらかじめ決められた答えはありません。

また、特定の技術や制度を前提とするものでもありません。

例えば、

- AIがすべて解決する
- 人間による音訳だけが望ましい
- 特定の電子書籍規格を採用すべき
- 日本独自の標準規格を直ちに作るべき

といった結論を前提にはしていません。

むしろ、それぞれの可能性と課題を整理しながら、現実的で持続可能な方法を探ることを目指します。

また、この分野では立場によって優先順位が異なることも自然なことだと考えています。

例えば、

- 当事者は「利用できること」を重視します。
- 出版社は「制作工程やコスト」を重視します。
- 研究者は「表現や理論」を重視します。
- エンジニアは「実装可能性」を重視します。
- 法律家は「権利や制度」を重視します。

これらは対立する考え方ではなく、それぞれ異なる視点から同じ課題を見ているものです。

本構想では、その違いを整理し、お互いの立場を理解しながら議論を進めることを大切にします。

# 当面の成果目標

ラウンドテーブルでは、すぐに一つの結論を出すことを目標にはしません。

当面の成果として、次のようなものを想定しています。

- 論点を整理したロードマップの作成
- 国内外の関連事例の収集
- マンガ表現とアクセシビリティに関する共通理解の形成
- 実証実験や共同研究のテーマ整理
- 出版・研究・技術分野を横断した継続的な対話の場づくり

これらは最終成果ではなく、今後の活動を進めるための土台となるものです。

# この文書について

本構想は、完成した提案書ではありません。

議論を重ねながら内容を更新し、必要に応じて修正・追加を行うことを前提とした作業文書です。

ロードマップやトピックノートとあわせて継続的に更新し、この文書自体をラウンドテーブルの活動記録として育てていくことを目指します。


# 今後の展望

この構想は、一度の会議や一つの研究で完結するものではありません。

マンガアクセシビリティというテーマは、技術、出版、研究、法律、教育、文化など、多くの分野が重なり合っています。

そのため、一つの専門分野だけで答えを出すことは難しく、継続的な対話と協力が必要になります。

当面は、小規模なラウンドテーブルから始めることを想定していますが、将来的には、より多くの専門家や関係団体が参加できる緩やかなネットワークへ発展することも期待しています。

その過程で、

- 共同研究
- 実証実験
- 標準化への提案
- 出版実務へのフィードバック
- 国際的な情報交換

など、さまざまな活動へ発展する可能性があります。

本構想は、その出発点となることを目指しています。

# おわりに

マンガアクセシビリティは、特定の技術だけで実現できるものではありません。

また、一人の専門家や一つの組織だけで取り組める課題でもありません。

それぞれ異なる立場の人たちが、自分の専門性を持ち寄り、お互いの考え方を理解しながら議論を重ねていくことが重要だと考えています。

この文書も、そのための「完成品」ではなく、「議論を始めるための土台」です。

今後、多くの方々から意見をいただきながら、この構想そのものも少しずつ育てていければと考えています。

どうぞよろしくお願いいたします。
