# マンガアクセシビリティ・ロードマップ

## この文書について

このロードマップは、現時点で考えられる論点を整理するための作業文書です。

完成した提案や標準仕様ではありません。

目的は、「どのような課題が存在し、それぞれの課題がどのようにつながっているのか」を見えるようにすることです。

今後のラウンドテーブルでは、この地図をもとに議論を進め、必要に応じて項目の追加や整理を行います。

また、それぞれのテーマについて、

- すでに取り組まれていること
- まだ十分に議論されていないこと
- 日本独自に検討が必要なこと
- 海外の動向を調査すべきこと

を整理するための基礎資料として利用することを想定しています。

# トップレベルの論点

現時点では、マンガアクセシビリティを次の九つのテーマに整理しています。

1. マンガ表現

2. 利用者・アクセシビリティ

3. 構造化・標準化

4. 音訳・解説

5. AI・技術

6. 法律

7. 倫理・文化

8. 出版・制作・流通

9. 推進体制・研究・資金

この九つは、それぞれ独立したテーマではなく、互いに深く関係しています。

例えば、AIによる音訳を考える場合でも、

マンガ表現の理解、

構造化データ、

法律、

出版実務、

利用者要求

などを同時に考える必要があります。

本ロードマップでは、それぞれのテーマを個別に整理しながら、相互の関係もあわせて確認していきます。



# 第一層　何を実現するのか

第一層では、「どのようなマンガアクセシビリティを目指すのか」という根本的なテーマを扱います。

ここでは、技術や制度よりも先に、「利用者が何を体験できるようになることを目標とするのか」を整理します。

## 1. マンガ表現

マンガは、文字だけでも画像だけでも成立するメディアではありません。

コマ割り、構図、視線誘導、時間表現、擬音、余白など、多様な表現が組み合わさって作品が成立しています。

検討項目の例

- コマ・ページ・見開きの役割
- 視線誘導と読み順
- コマ間・コマ内の時間表現
- 構図や演出の言語化
- 作家・作品・ジャンルごとの表現の違い
- 「読む」と「鑑賞する」の違い

## 2. 利用者・アクセシビリティ

アクセシビリティの目的は、「読めるようにすること」だけではありません。

利用者が作品を理解し、自分なりに楽しめることも重要な目標になります。

また、利用者の状況によって必要とする支援は異なります。

検討項目の例

- 全盲・弱視など利用者の違い
- 先天盲と中途失明による違い
- 詳細説明と簡潔な説明の選択
- 利用者が情報量を調整できる仕組み
- 「利用可能」と「理解しやすい」の違い
- アクセシビリティと鑑賞体験の両立

## 3. 音訳・解説

作品を言葉で伝える方法を整理します。

単なる画像説明ではなく、マンガという表現に適した解説とは何かを検討します。

検討項目の例

- ページ単位・見開き単位・コマ単位の使い分け
- セリフと情景説明の関係
- 登場人物や話者の表現方法
- 擬音・描き文字の扱い
- 演出や感情をどこまで説明するか
- 人間による音訳とAI生成の役割分担



# 第二層　どう実現するのか

第二層では、第一層で整理した目標を、どのような仕組みや技術、制度によって実現するのかを整理します。

ここでは、マンガアクセシビリティを社会の中で継続的に実現するための基盤を扱います。

## 4. 構造化・標準化

マンガの内容や構造を、どのようなデータとして記録・共有するかを整理します。

国内だけでなく、海外で提案されている標準仕様との関係も重要なテーマとなります。

検討項目の例

- ページ・見開き・コマなどの管理単位
- 読み順・視線誘導の記録方法
- コマや吹き出しなどの構造情報
- EPUB、Readium、DiViNaなど既存仕様との関係
- 日本のマンガ表現との適合性
- 国際標準への提案やフィードバック

## 5. AI・技術

AIや画像認識技術を、どのように活用するかを整理します。

AIは重要な技術ですが、それだけで課題を解決できるものではありません。

人間との役割分担も重要になります。

検討項目の例

- コマ・人物・吹き出しの認識
- 読み順の推定
- 登場人物や話者の識別
- 作品文脈の理解
- AIによる解説生成
- 人間による確認・修正
- 閲覧アプリとの連携

## 6. 法律

アクセシビリティを実現するためには、権利処理や制度についても整理する必要があります。

利用者、出版社、著作者、それぞれの立場を踏まえた検討が求められます。

検討項目の例

- 著作権
- 著作者人格権
- 障害者向け提供に関する制度
- AI利用に伴う権利処理
- 解説文や音訳データの権利
- 出版社・配信事業者との役割分担

## 7. 倫理・文化

法律だけでは整理できない課題も数多く存在します。

マンガ文化そのものを尊重しながら、多様な価値観や社会的な受け止め方について議論することも重要です。

検討項目の例

- 表現の自由
- 性的表現・暴力表現
- 年齢表現と文化差
- AIによる解説の倫理
- 二次創作・同人文化
- 海外展開における文化的な違い
- 多様な作品解釈の尊重

## 8. 出版・制作・流通

アクセシビリティを社会へ普及させるためには、制作工程や流通の仕組みも重要になります。

理想論だけではなく、現場で実現できる方法を考える必要があります。

検討項目の例

- 制作工程への組み込み
- 人手とAIの役割分担
- 品質管理
- 制作コスト
- 出版社・配信事業者・制作会社の役割
- 既刊作品への対応
- アクセシビリティ情報の流通
- 継続可能な運用体制



# 第三層　誰が進めるのか

第三層では、マンガアクセシビリティを継続的に推進するための体制について整理します。

このテーマは、一人の研究者や一つの企業だけで進められるものではありません。

異なる専門分野や組織が、それぞれの役割を理解しながら協力できる仕組みを考える必要があります。

## 9. 推進体制・研究・資金

マンガアクセシビリティは、研究だけでも、出版だけでも実現できません。

継続的な研究、実証実験、社会実装を進めるためには、分野を横断した協力体制が必要です。

検討項目の例

- ラウンドテーブルの運営
- 出版社・研究者・当事者・エンジニアの連携
- 学会・業界団体との協力
- 実証実験の企画
- 研究テーマの整理
- 国内外の情報共有
- 人材育成
- 研究費・助成金・共同研究
- 継続的な運営体制
- 社会実装までを見据えたロードマップ

# 三層構造の整理

本ロードマップでは、マンガアクセシビリティを次の三層で整理します。

## 第一層　何を実現するのか

- マンガ表現
- 利用者・アクセシビリティ
- 音訳・解説

ここでは、「どのような読書体験・鑑賞体験を実現したいのか」を整理します。

## 第二層　どう実現するのか

- 構造化・標準化
- AI・技術
- 法律
- 倫理・文化
- 出版・制作・流通

ここでは、「その目標をどのような技術・制度・運用によって実現するのか」を整理します。

## 第三層　誰が進めるのか

- 推進体制・研究・資金

ここでは、「誰が、どのような体制で、この取り組みを継続していくのか」を整理します。

# このロードマップの使い方

この文書は、完成した提案ではありません。

議論を始めるための「地図」です。

今後、それぞれのテーマについて専門家から意見をいただきながら、

- 新しい項目を追加する
- 項目同士の関係を整理する
- 優先順位を見直す
- 実証実験や共同研究へ反映する

ことを想定しています。

ロードマップそのものも、ラウンドテーブルの活動を通して継続的に更新していく予定です。
